# ☕ Starbucks Beverage Nutrition Analysis Dashboard

## Overview

Consumers today are increasingly focused on understanding the nutritional value of the products they consume. Starbucks offers a wide variety of beverages with different calorie, sugar, and caffeine levels, making it difficult to compare options and make informed choices.

This Power BI dashboard transforms Starbucks beverage nutrition data into actionable insights, allowing users to explore nutritional trends, compare beverage categories, and identify healthier drink options.

---

## Business Problem

Customers often struggle to evaluate the nutritional impact of Starbucks beverages due to the large variety of products available.

Questions addressed by this project include:

* Which beverage categories contain the highest calories?
* Which drinks have the highest sugar content?
* How does caffeine vary across products?
* Which beverage categories offer healthier alternatives?

---

## Project Objectives

* Analyze nutritional information across Starbucks beverages.
* Compare calorie, sugar, and caffeine levels.
* Identify high-calorie and high-sugar beverages.
* Provide an interactive dashboard for data exploration.
* Support data-driven nutritional awareness.
